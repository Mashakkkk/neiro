# Нейро

Минималистичный адаптивный лендинг о работе с искусственным интеллектом. Проект показывает, как ИИ помогает развивать идеи, сокращать рутину и принимать решения.

## Возможности

- адаптивная вёрстка для компьютеров и мобильных устройств;
- пастельная дизайн-система без внешних изображений;
- семантическая HTML-разметка и доступная навигация;
- плавные переходы по разделам с поддержкой `prefers-reduced-motion`;
- готовая production-сборка для Cloudflare Workers через vinext.

## Технологии

- Next.js 16 и React 19;
- TypeScript;
- CSS и Tailwind CSS;
- vinext / Vite.

## Запуск локально

Требуется Node.js 22.13 или новее.

```bash
npm install
npm run dev
```

Проверка production-сборки:

```bash
npm run build
```

## Структура

- `app/page.tsx` — содержание и компоненты лендинга;
- `app/globals.css` — дизайн-система и адаптивные стили;
- `app/layout.tsx` — метаданные и корневой layout.
